
- # Precious Metals Data Platform

An end-to-end demo of a **data pipeline for precious metals trading**, built with Python and Streamlit.  
This project simulates a real-world ETL and BI workflow: from **data generation → ETL → SQLite warehouse → interactive dashboard**.

---

## **Project Overview**

This project demonstrates skills in:  
- Data generation and simulation of trading data  
- Building ETL pipelines to transform and store data  
- Creating a relational warehouse using SQLite  
- Visualizing data with an interactive Streamlit dashboard  

The dashboard allows users to explore:
- Trade dataset preview  
- Daily trade counts  
- Metal breakdown by type  

---

## **Features**

- ✅ **Synthetic data generation**: 50 customers, 4 metals, 3,000 trades  
- ✅ **ETL pipeline**: reads CSVs → creates staging tables → builds fact table  
- ✅ **SQLite warehouse**: stores structured trading data  
- ✅ **Streamlit dashboard**: interactive charts and tables  
- ✅ **Fully self-contained**: no external data required, works on Streamlit Cloud and iPad
